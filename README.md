# English Grammar Game – Infinite Levels

An interactive web application to master English grammar through level-based assessments.  
Users sign in with Firebase, track their progress, and challenge themselves from beginner to advanced levels.

## Features

- 🔐 **Authentication** – Sign up / Sign in with email & password via Firebase.
- 📊 **Dashboard** – View levels finished, average score, total marks, and next level.
- 📈 **Personalized Feedback** – Identifies weak topics based on performance.
- 🎯 **Level-Based Assessments** – 10 levels covering Parts of Speech, Tenses, Modals, Conditionals, Passive Voice, and Advanced Grammar.
- 📱 **Responsive UI** – Works seamlessly on mobile, tablet, and desktop.
- 🔄 **Progress Persistence** – All data stored in Firestore; reset option available.

## Tech Stack

- HTML5 / CSS3 / JavaScript (vanilla)
- Firebase Authentication
- Firebase Firestore (NoSQL database)
